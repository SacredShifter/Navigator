# Sacred Shifter Module Recommendation System

**Version:** 1.0.0
**Purpose:** Guide Cursor AI and Sacred Shifter system in recommending appropriate healing modules based on Navigator profile results

---

## üéØ Overview

After a user completes the Navigator assessment, they receive a **consciousness profile** that determines which Sacred Shifter modules and experiences will best support them. This document details the recommendation logic for Cursor AI to implement.

### Core Recommendation Principles

1. **Safety First** - Never recommend experiences beyond user's nervous system capacity
2. **Chemical State Awareness** - Adjust recommendations based on substance/medication use
3. **Regulation Matching** - Match intensity to current regulation level
4. **Profile Alignment** - Honor the user's archetypal consciousness state
5. **Progressive Deepening** - Start gentle, increase challenge as capacity builds

| Profile | Chemical State | Override Action | Rationale |
|---------|---------------|-----------------|-----------|
| **Lost** | Any | ‚Üí Trauma Safety Track | Core trauma present |
| **Lost** | Psychedelic | ‚Üí Grounding Protocol ONLY | High crisis risk |
| **Lost** | Withdrawal | ‚Üí Recovery Track | Medical stabilization needed |
| **Awakening** | Psychedelic | ‚Üí Grounding Protocol | Reality anchoring critical |
| **Awakening** | Withdrawal | ‚Üí Recovery Track | Stabilize before awakening work |
| **Any** | Withdrawal + Low Regulation | ‚Üí Recovery Track | Medical priority |
| **Numb** | Psychedelic | ‚Üí Gentle Grounding | Can't integrate without felt sense |
| **Expander** | Stimulant | Moderate all intensity | Mania/burnout risk |

### Safety Gate Implementation

```typescript
function applySafetyOverrides(
  profile: NavigatorProfile,
  chemicalState: ChemicalState,
  regulationLevel: RegulationLevel
): { recommendedTrack: string; reason: string } {

  // Lost + any psychedelic = immediate grounding
  if (profile.name === 'Lost' && chemicalState === 'psychedelic') {
    return {
      recommendedTrack: 'grounding-protocol',
      reason: 'You\'re in a vulnerable state while experiencing altered consciousness. Let\'s prioritize grounding and safety first.'
    };
  }

  // Any profile + withdrawal + low regulation = recovery
  if (chemicalState === 'withdrawal' && regulationLevel === 'low') {
    return {
      recommendedTrack: 'recovery-track',
      reason: 'Your nervous system needs stabilization support right now. Let\'s focus on recovery first.'
    };
  }

  // Awakening + psychedelic = grounding priority
  if (profile.name === 'Awakening' && chemicalState === 'psychedelic') {
    return {
      recommendedTrack: 'grounding-protocol',
      reason: 'Your consciousness is expanding. Let\'s make sure you stay anchored and oriented.'
    };
  }

  // Continue for all override scenarios...
}
```

## üîÑ Reassessment Triggers

Recommend Navigator reassessment when:

1. **30+ days since last assessment** (natural state shifts)
2. **Major life event** (user reports change)
3. **Chemical state change** (started/stopped substances)
4. **User expresses misalignment** ("This doesn't feel right")
5. **Safety concerns arise** (repeated distress signals)
6. **Profile completion** (e.g., Integrator completes integration, ready to expand)

### Cursor AI Reassessment Prompt

```
"I notice it's been [X weeks] since your last Navigator assessment.
Things may have shifted for you - that's natural.

Would you like to retake the assessment? It only takes 5-7 minutes,
and it helps us make sure we're recommending experiences that truly
serve where you are *now*.

You can also explore without reassessing - your choice entirely."
```
